# 杀戮尖塔 2 选牌助手 (STS2 Card Advisor)

基于 [blackpatton17/sts2-draw-rate](https://github.com/blackpatton17/sts2-draw-rate) 开发的增强版选牌推荐 Mod。原版仅显示静态胜率/选取率统计，本版本新增了**流派感知的动态推荐系统**。

## 核心功能

### 1. 流派检测与成型度

实时分析牌组，自动识别 15 种流派，并计算**流派成型度百分比**。成型度基于已拥有的标志牌和核心牌占比，协同牌提供额外加成。

### 2. 选牌推荐（三选一 / 商店）

- **三维度评分** — 流派协同、统计数据、牌组需求综合打分
- **阶段动态权重** — 前期侧重生存需求，后期侧重流派协同
- **模拟选牌** — 评分基于"选了这张牌之后"的牌组状态
- **三选一对比** — 全部偏弱时标出"三选一最优"
- **血量感知** — 低血量时提升防御牌权重
- **商店性价比** — 商店中考虑金币花费

### 3. 牌组查看模式

打开牌组界面时自动切换为**适配度评估**模式：

- **适配度标注** — 每张牌按流派适配度五档颜色标注（核心/重要/适配/可精简/建议删除）
- **删除建议** — 基础Strike/Defend标记"建议删除"，与流派冲突的牌也会标记
- **锻造优先级** — hover 显示锻造建议（★★★优先 / ★★☆推荐 / ★☆☆可以）
- **总体建议** — hover 显示牌组总览（流派成型度、缺陷分析、下一步建议）

### 4. 智能过滤

- 战斗中自动隐藏评分框
- 诅咒牌/状态牌/Token牌不显示评分
- 选牌界面隐藏基础Strike/Defend

## 评级说明

**选牌模式：**

| 颜色 | 星级 | 含义 |
|------|------|------|
| 🟠 橙 | ★★★★★ | 极力推荐 |
| 🟣 紫 | ★★★★☆ | 推荐 |
| 🔵 蓝 | ★★★☆☆ | 可选 |
| 🟢 绿 | ★★☆☆☆ | 可跳 |
| ⚪ 灰 | ★☆☆☆☆ | 不推荐 |

**牌组查看模式：**

| 颜色 | 含义 |
|------|------|
| 🟠 橙 | 核心牌 — 流派标志，不可删除 |
| 🟣 紫 | 重要 — 流派关键组件 |
| 🔵 蓝 | 适配 — 与流派有协同 |
| 🟢 绿 | 可精简 — 与流派关系不大 |
| ⚪ 灰 | 建议删除 — 基础牌/冲突牌 |

## 安装

### 前置条件
- 《杀戮尖塔 2》Steam 版，切换到 **Public Beta** 分支
- 游戏内启用 Mod 加载（首次会弹出确认窗口）

### 安装步骤

1. 从 [Releases](../../releases) 下载最新版本
2. 解压到游戏的 `mods` 目录：
   - **Windows**: `Slay the Spire 2/mods/CardProbMod/`
   - **macOS**: `SlayTheSpire2.app/Contents/MacOS/mods/CardProbMod/`
3. 目录结构：
   ```
   mods/
   └── CardProbMod/
       ├── CardProbMod.dll
       ├── CardProbMod.json
       └── result_cleaned.csv
   ```
4. 启动游戏，在选牌/商店界面即可看到推荐

> **注意**: 不需要 BaseLib，游戏原生支持 Mod 加载。

## 开发

```bash
# 需要 .NET 9.0 SDK
# lib/ 目录下需要 0Harmony.dll, GodotSharp.dll, sts2.dll

# 编译
dotnet build Sts2Mod.csproj -c Release

# macOS 一键编译部署
./deploy.sh
```
